💰 無痛存錢法管家 (Smart Money Manager)
這是一款專為現代人設計的「無記帳負擔」個人理財 Web App。結合了「50/30/20 黃金比例」、「虛擬信封理財法」以及「目標進度追蹤」，幫助您在不影響生活品質的前提下，輕鬆管理債務並累積財富。

👉 點擊這裡立即開啟 App 開始記帳！
✨ 核心特色與功能
🛡️ 極致隱私，免登入：完全純前端架構 (Vanilla JS)，您的所有財務數據與歷史紀錄皆加密儲存於您設備的本地端 (Local Storage)，無任何資料外洩風險。

📱 PWA 支援 (支援加入主畫面)：可用手機瀏覽器開啟並選擇「加入主畫面」，即可獲得如原生 App 般的全螢幕沉浸式體驗，甚至支援離線操作。

🤖 智慧比例分配：自動計算總收入扣除「固定支出」與「負債」後的可用結餘，並可一鍵套用系統建議的黃金分配比例（預算本 50% / 預存本 20% / 存錢本 30%）。

💳 信用卡無痛收納袋：專為信用卡族群設計！記帳時選擇「刷卡」，系統會自動將該筆金額移入「卡費收納袋」，結帳日不再心痛。

🎯 視覺化目標追蹤：自訂您的「願望清單」與「存錢目標」，進度條即時顯示。結算後的結餘可彈性分配至各個小目標。

🕰️ 雲端/本地歷史封存：每個月底一鍵「執行月結算」，當月所有收支明細將會被封存成唯讀歷史紀錄，隨時可回顧。

📦 支援資料匯出 / 匯入：內建.json 備份機制，換手機或清空瀏覽器也不怕資料遺失。

🛠️ 技術堆疊 (Tech Stack)
HTML5 & CSS3

JavaScript (ES6+)：純前端邏輯與 DOM 操作，無依賴大型框架。

Tailwind CSS：透過 CDN 引入，實現現代化且響應式的 UI 設計。

Chart.js：實現動態資金分配圓餅圖。

Firebase (可選)：預留雲端 Firestore 備份串接點。

💡 使用指南 (How to Use)
盤點與倒推：設定您的當月收入，並扣除房租、信貸等「固定支出與負債」。

預算分配：透過視覺化儀表板，手動微調或自動分配三大帳本（預算、預存、存錢）的額度。

日常記帳：在「預算本」中記錄日常開銷，刷卡消費會自動幫您把現金移至卡費袋。

月末結算：月底點擊結算，將本月未花完的結餘匯入水庫，並將其分配給您的「預存願望」或「存錢目標」中！
